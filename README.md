# MyDevArchive

A personal development portfolio & certificate management system to upload, organize, and showcase your course certificates, badges, contribution recognitions, and offer letters.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, MUI
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **File Storage:** Cloudinary
- **Version Control:** Git, GitHub

## Features

- Upload and manage **course certificates** with issuer, category, and image.
- Upload and manage **badges** earned from competitions or courses.
- Store **contribution certificates** and **badges** from open-source or hackathons.
- Keep track of **internship and offer letters**.
- REST API powered backend for CRUD operations.
- Cloudinary integration for image hosting.
